\begin{abstract}
Proflog is a relational implementation of a first-order logic programming
language and proof kernel built in the miniKanren tradition. This paper
describes its implementation strategy, its rationale, and the engineering
lessons learned from making theorem proving, proof search, and answer synthesis
available as relational computations.
\end{abstract}
